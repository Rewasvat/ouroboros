Tema da Monografia

Sistema C++ para integração com linguagens de script.


Resumo

É comum, principalmente no desenvolvimento de jogos, o desenvolvimento de uma aplicação em C++ que, entre outras coisas, incorpora e executa scripts em alguma linguagem como Lua ou Python para facilitar diversos aspectos do desenvolvimento e permitir alteração ou introdução de lógica na aplicação por outras pessoas sem necessitar recompilar o aplicativo.

No entanto, tal incorporação de C++ e uma linguagem de script pode trazer vários desafios, como escolha de qual linguagem usar e aprender/usar a API da linguagem para incorporá-la. Esse trabalho consiste no desenvolvimento de um sistema em C++ fácil de usar que permite ao usuário incorporar facilmente diversas linguagens de script em seu programa.


Objetivos

  • Desenvolvimento de uma biblioteca C++ que abstrai o uso da API de uma(s) linguagem(s) de script(s) de forma clara, fácil de usar, e facilmente estendida para aceitar outras linguagens de script, sendo que inicialmente terá suporte a Lua e Python. O usuário da biblioteca não necessitará de conhecimento da API da linguagem para usar tal linguagem em seu programa, e o programa não precisa se importar em qual linguagem ele está usando, dado que seus scripts sejam feitos em alguma linguagem que o sistema reconheçe.
  • Desenvolvimento de uma ferramenta que parseia código C++ (uma API do usuário) a fim de gerar wrappers dessa API para cada linguagem de script que o sistema reconheçe. Para que ela possa ser usada pelos scripts que a biblioteca executa.

Atividades Já Realizadas

  • A biblioteca já existe, porém ainda depende de uma ferramenta externa (SWIG) para geração dos wrappers que introduz vários problemas na incorporação das linguagens;
  • Versão preliminar de um parser C++ para geração dos wrappers (para substituir o SWIG);
